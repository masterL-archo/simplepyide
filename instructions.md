  Step 1: Download the code , or clone it using ```git clone https://github.com/masterL-archo/simplepyide```
  Step 2: Download Python from the official site https://www.python.org/downloads    
  Step 3: Go to folder and run `pip install -r requirements.txt` to download required modules.    
  Step 4: Go to /src/ and run `python main.py`.   
  Step 5: Have fun, if you find any bugs/issues feed it back.
